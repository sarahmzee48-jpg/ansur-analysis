## Project Overview
This project analyzes the ANSUR II dataset to explore anthropometric and demographic characteristics of U.S. Army personnel.

## Contents
- Project report (Word document)
- Stata do-file
- (Optional) Markdown output

## Methods
Data cleaning, descriptive statistics, stratified analysis, correlation analysis, and BMI calculations were conducted using Stata.

## Dataset
ANSUR II (2012 U.S. Army Anthropometric Survey)
